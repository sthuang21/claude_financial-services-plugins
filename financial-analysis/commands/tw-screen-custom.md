# /tw-screen-custom [條件描述]

自訂條件選股，從上市股票中依照使用者指定條件篩選。

## 使用範例
/tw-screen-custom 殖利率大於6%且外資近10日買超且本益比低於15倍
/tw-screen-custom ROE大於20%且月營收連續成長且股價在年線之上
/tw-screen-custom 週KD低檔交叉且投信連買3日且市值大於100億

## 執行流程
1. 解析使用者輸入的條件
2. 對應到可用的 FinMind 資料集
3. 依條件逐一篩選上市股票
4. 回傳符合全部條件的股票清單

## 支援條件關鍵字
- 基本面：殖利率、本益比、ROE、EPS、營收、毛利率、負債比
- 技術面：日KD、週KD、月KD、月線、季線、年線、均線、突破、成交量
- 籌碼面：外資、投信、自營商、融資、融券、大戶持股
- 市值條件：市值大於/小於 N 億

⚠️ 本選股結果由 AI 運算產出，僅供參考，不構成投資建議。
