# 台股技術分析

## FinMind 可取得的技術面資料集
- TaiwanStockPrice：日K歷史股價（開高低收量）
- TaiwanStockPriceAdj：還原權值股價
- TaiwanStockPER：本益比、本淨比歷史
- TaiwanStockMinuteData：分K資料

## 常用指標
- KD：K<20 超賣，K>80 超買
- MACD：用於確認趨勢轉折
- 均線：5MA、10MA、20MA、60MA、120MA、240MA
- 布林通道：±2 標準差

## 台股特有觀察重點
- 週KD 比日KD 更重要，用於判斷中期趨勢
- 季線（60MA）：法人平均成本線，常為支撐壓力
- 月線（240MA）：長期多空分界

## 標準分析步驟
1. 月線圖確認大趨勢方向
2. 週線確認中期位置與KD狀態
3. 日線找進出場時機
4. 成交量確認訊號有效性
