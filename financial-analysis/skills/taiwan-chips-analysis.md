# 台股籌碼分析

## FinMind 可取得的籌碼面資料集
- TaiwanStockInstitutionalInvestorsBuySell：三大法人買賣超
- TaiwanStockMarginPurchaseShortSale：個股融資融券
- TaiwanStockShareholding：外資持股比例
- TaiwanStockHoldingSharesPer：股權分散表
- TaiwanDailyShortSaleBalances：借券成交明細
- TaiwanStockTradingDailyReport：分點券商資料（主力追蹤）

## 三大法人判讀
- 外資：最具影響力，連續買超 5 日以上為強勢訊號
- 投信：著重季底作帳行情（3、6、9、12月）
- 自營商：短線為主，方向參考性較低

## 融資融券判讀
- 融資增加但股價不漲 → 警訊，籌碼凌亂
- 融券高 + 股價強 → 軋空題材機會
- 融資使用率 >70% → 高風險警示

## 分析流程
1. 查三大法人近 20 日累計買賣超
2. 確認融資融券餘額趨勢
3. 查股權分散表，確認大戶持股是否增加
4. 有需要時查分點資料追蹤主力動向
